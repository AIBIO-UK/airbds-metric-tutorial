# Further Resources

## About the AIRBDS Working Group

The AIRBDS metric was developed by the **AI-Ready Bioscience Datasets (AIRBDS) Working Group** of [AIBIO-UK](https://aibio.ac.uk), funded by the Biotechnology and Biological Sciences Research Council (BBSRC).

- Working group page: [aibio.ac.uk/about/working-groups/airbds/](https://aibio.ac.uk/about/working-groups/airbds/)
- Repository: [github.com/AIBIO-UK/airbds-metric](https://github.com/AIBIO-UK/airbds-metric)

---

## Contributing reviews

Dataset reviews submitted to the repository help build a public record of AI-readiness across the bioscience landscape. You are very welcome to contribute.

- Contribution guidelines: [CONTRIBUTING.md](https://github.com/AIBIO-UK/airbds-metric/blob/main/CONTRIBUTING.md)
- Code of Conduct: [CODE_OF_CONDUCT.md](https://github.com/AIBIO-UK/airbds-metric/blob/main/CODE_OF_CONDUCT.md)

## FAIR Data Principles

The AIRBDS metric builds on several established frameworks for data quality and openness:

- **FAIR Principles** — Wilkinson et al. (2016). *The FAIR Guiding Principles for scientific data management and stewardship.* Scientific Data, 3, 160018. [doi:10.1038/sdata.2016.18](https://doi.org/10.1038/sdata.2016.18)
- **ESIP Data Quality Cluster** — [wiki.esipfed.org](https://wiki.esipfed.org)
- **Open Data Institute (ODI) Certificates** — [certificates.theodi.org](https://certificates.theodi.org)
- **Bridge2AI Standards** — [bridge2ai.org](https://bridge2ai.org)
